# Presentation Outline

1. Problem: AI-assisted code changes can leave unclear evidence about what was actually validated.
2. Promise: Tracepack creates a local redacted review bundle for human review.
3. Demo: show missing-validation and corrected reports.
4. Privacy: no source upload, no prompts, no transcripts, no `.env`, no credentials.
5. Honesty: no correctness, safety, security, or approval claims.
6. Architecture: TypeScript CLI, local Git, user-approved commands, static HTML.
7. Next step: validate with maintainers and small-team developers using real PR workflows.
